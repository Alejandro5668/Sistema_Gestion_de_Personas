# Gestión de Personas 👥

Este proyecto es un **CRUD (Create, Read, Update, Delete)** desarrollado en **PHP, MySQL, HTML y Bootstrap**, enfocado en la gestión de personas.

## 🚀 Funcionalidades
- Registrar personas
- Visualizar registros
- Modificar información
- Eliminar usuarios

## 🛠️ Tecnologías
- PHP
- MySQL
- HTML5
- Bootstrap
